Для выхода из программы введите "end" без кавычек во входной терминал

Программа способна считать любую строку без пробелов, табуляций и прочих подобных символов
Это дает возможность передавать сразу несколько пакетов одним нажатием Enter :)
После считывания входных данных они перобразуются в байтовый вид
Далее происходит парсинг, передача и обработка
При выводе строка преобразуется обратно в в hex формат, нули в начале сохранены

Программа использует протокол TCP для коммуникации между потоками
IP - localhost
Port - выбирается первый незанятый от 7090 до 7099 включительно
Потоки подключаются к серверу на основном потоке, и через него происходит обмен данными
